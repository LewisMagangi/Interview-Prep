# Interview-Prep

A comprehensive collection of resources and study guides for technical interview preparation.

## Overview

This repository contains various roadmaps and study materials to help you prepare for software engineering interviews, focusing on data structures, algorithms, system design, and more.

## Contents

### Roadmap Directory

- **pattern.md**: Complete DSA patterns study guide with tagged problems from LeetCode, Codeforces, GeeksforGeeks, HackerRank, and others.
- **datastructures-and-algorithms.pdf**: DSA roadmap
- **system-design.pdf**: System design guide
- **computer-science.pdf**: Computer science fundamentals
- **api-design.pdf**: API design principles
- **backend.pdf**: Backend development roadmap
- **frontend.pdf**: Frontend development roadmap
- **devops.pdf**: DevOps roadmap
- **javascript.pdf**: JavaScript roadmap
- **python.pdf**: Python roadmap
- **react.pdf**: React roadmap
- **nextjs.pdf**: Next.js roadmap
- **nodejs.pdf**: Node.js roadmap
- **linux.pdf**: Linux roadmap
- **software-architect.pdf**: Software architect guide
- **software-design-architecture.pdf**: Software design and architecture
- **roadmap.md**: General roadmap overview
- **dsa_questions_per_topic**: Additional DSA questions organized by topic

## Getting Started

1. Clone the repository:

   ```bash
   git clone https://github.com/LewisMagangi/Interview-Prep.git
   ```

2. Navigate to the roadmap directory and explore the materials.

3. Use the tagged problems in `pattern.md` to practice on the respective platforms.
